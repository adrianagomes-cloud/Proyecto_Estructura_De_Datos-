Este proyecto consiste en una herramienta de gestión y análisis de redes neuronales diseñada para modelar la conectividad sináptica del cerebro humano. 
El sistema permite visualizar las conexiones entre neuronas, identificar zonas aisladas tras una posible lesión y calcular la eficiencia en 
la transmisión de señales eléctricas bajodiversas condiciones químicas.
